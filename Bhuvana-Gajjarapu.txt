13. Cherry-pick Practice
I learned how to copy one specific commit from one branch to another branch without merging the full branch.

Commands:
- git log --oneline
- git checkout main
- git cherry-pick <commit-hash>
- git status
- git push

14. Difference Between Fetch and Pull (Implemented)
I understood and implemented the practical difference:
- git fetch only downloads new changes from remote and does not change my working files.
- git pull downloads changes and merges/rebases them into my current branch.

Commands I used:
- git fetch origin
- git status
- git log
- git pull origin main

15. Difference Between Reset and Rebase
I learned the purpose of both commands and where to use them:
- git reset is used to move HEAD backward and undo local commits or staging.
- git rebase is used to replay my branch commits on top of another base to keep cleaner history.

Commands I learned:
- git reset --soft HEAD~1
- git reset --mixed HEAD~1
- git rebase origin/main
- git rebase --continue

16. Git Stash Learning
I learned how to temporarily save my uncommitted work using stash and restore it later.

Commands I learned:
- git stash
- git stash list
- git stash apply
- git stash pop
- git stash drop
- git stash clear
